IMPORTANT

When importing pixel art and music, click the blank space like on this text so that it prompts your web browser to open your file explorer.

Steps

  1. Import music: MUST BE .ogg file, you can convert it online.
  2. Import pixel art: MUST BE .png FILE AND MUST BE 16x16 pixels; please size it down to 16x16 pixels or else it won't work.
  3. Enter BPM: Use this to find the BPM of the song; must enter BPM or the editor will break
  4. Left click to add a note, right click to remove. Use the undo/redo buttons to revert changes.
  5. To add pauses or silent notes, just click anywhere outside the 16x16 frame and it will treat it as a skipped note.
  6. Use the Preview button to simulate the current beatmap
  7. Use the export button and it will copy the beatmap information onto your clipboard; paste it to the discord to share it for others to import.
